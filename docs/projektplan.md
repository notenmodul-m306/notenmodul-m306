# Phase 3 – Projektplan

## Vorgehensmodell
Für das Projekt wird ein agiles Vorgehensmodell in vereinfachter Form (Scrum-light) gewählt.

### Begründung
Das Team ist klein und arbeitet iterativ. Ein agiles Vorgehen ist deshalb sinnvoll, weil:
- Anforderungen während des Projekts konkretisiert werden können
- Fortschritte regelmässig überprüft werden
- Aufgaben flexibel priorisiert werden können
- Probleme früh erkannt und angepasst werden können
